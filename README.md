# Gateway - zuul

Servicio que serive como puerta de entrada para los microservicios.

## Api

- http://localhost:8090/

## Tecnología de desarrollo
- [Spring-boot ](https://spring.io/projects/spring-boot)
- Java 11
- Maven
- IntelliJ IDEA

## Instalación

```bash
./mvnw clean
./mvnw install
./mvnw spring-boot:run
```
